# Driver

Simple Framework for API Client.

Client, Model, APIを分離するアイデアです.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'driver'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install driver

## Usage

```ruby
require 'driver'
```
